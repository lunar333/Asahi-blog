## 二分搜索
## 题目链接：[704二分查找](https://leetcode.cn/problems/binary-search/)

```python
class Solution:
    def search(self, nums: List[int], target: int) -> int:
        left = 0
        right = len(nums)-1
        while(left <= right):
            middle = (left+right)// 2
            if (target< nums[middle]):
                right = middle - 1
            elif (target > nums[middle]):
                left = middle + 1
            else:
                return middle
        return -1
```
